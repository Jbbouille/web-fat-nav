# web-fat-nav

It helps me to navigate without mouse (so quickly) through the web.

## How to use:

1. Tap `ctrl`.
2. Enter a letter corresponding to the link you want to go (example: `ze`).
3. Tap `Enter` or remove a letter with `alt` + `<-`.
